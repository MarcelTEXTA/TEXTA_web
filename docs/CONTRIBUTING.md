# Contributing to TEXTA web

TEXTA web is an open-source project, and we encourage contributions from the community to improve the browser and add new features. If you would like to contribute to the development of TEXTA web, please follow the guidelines below.

## AGPL-3.0 License

TEXTA web is distributed under the AGPL-3.0 license. This means that any contribution must be compatible with this license. Please make sure you understand the implications of this license before submitting your contribution.

## How to Contribute

### If you have GitHub

1. Fork the TEXTA web repository.
2. Clone your fork locally.
3. Create a new branch for your contribution.
4. Make your changes and ensure they are properly tested.
5. Commit your changes with a clear and descriptive message.

### If you do not have GitHub

1. Download the TEXTA web source code from the GitHub repository.
2. Make your changes locally.
3. Contact TEXTA support through the appropriate channels listed in the GitHub repository to submit your contribution.

## Contacting TEXTA Support
