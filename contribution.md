# Guide pour la contribution à TEXTA web

TEXTA web est un projet open-source, et nous encourageons les contributions de la communauté pour améliorer le navigateur et ajouter de nouvelles fonctionnalités. Si vous souhaitez contribuer au développement de TEXTA web, voici quelques directives à suivre :

## Licence AGPL-3.0

TEXTA web est distribué sous la licence AGPL-3.0, ce qui signifie que toute contribution doit être compatible avec cette licence. Assurez-vous de comprendre les implications de cette licence avant de soumettre votre contribution.

## Comment contribuer

### Si vous avez GitHub :

1. Forkez le dépôt GitHub de TEXTA web.
2. Clonez votre fork localement.
3. Créez une nouvelle branche pour votre contribution.
4. Apportez vos modifications et assurez-vous de les tester.
5. Commitez vos changements avec un message clair et descriptif.

### Si vous n'avez pas GitHub :

1. Téléchargez le code source de TEXTA web depuis le dépôt GitHub.
2. Apportez vos modifications localement.
3. Contactez le support TEXTA via les canaux appropriés indiqués dans le dépôt GitHub pour soumettre votre contribution.

## Contacter le support TEXTA

Pour toute question ou support concernant la contribution à TEXTA web, veuillez contacter le support TEXTA via les canaux appropriés indiqués dans le dépôt GitHub. Nous sommes là pour vous aider et répondre à vos questions via :
