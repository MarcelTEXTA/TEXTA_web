## Overview (coming soon)

TEXTA web will be an open-source Chromium-based browser designed to provide a secure and anonymous browsing experience while maintaining smooth and fast navigation. This browser focuses on user privacy by integrating advanced tracker and advertisement blocking features, while ensuring optimal compatibility with modern websites.

TEXTA web will be ideal for users who care about their privacy and want to browse the Internet without being tracked by third parties. With its user-friendly interface and customization options, TEXTA web allows users to configure their browsing experience according to their preferences while benefiting from enhanced protection against online threats.

This browser is also designed to be lightweight and efficient, minimizing the impact on system resources while providing high performance. In summary, TEXTA web is a secure and privacy-respecting browsing solution adapted to the needs of modern users.

Learn more on the [project wiki](https://github.com/MarcelTEXTA/TEXTA_web/wiki).

## Main Features

- **Anonymous browsing**: Blocks trackers and advertisements to protect user privacy.
- **Chromium browser with CefSharp**: TEXTA web uses a Chromium engine to ensure optimal compatibility with modern websites and provide a smooth and fast browsing experience.
- **New interface for a modern browser**: Tabs are displayed on the left and use workspaces to organize browsing sessions.
- **Advanced customization**: Allows users to configure privacy and security settings according to their preferences.
- **Lightweight and efficient**: Designed to minimize system resource usage while delivering high performance.
- **Windows-only support**: This application is developed using WPF in C# and is only compatible with Windows operating systems.

## Security Warning and Limitations

TEXTA web is not yet complete. This means that some features are not available or may not work as expected, and anonymous browsing cannot be guaranteed.

It is recommended not to use this browser for daily activities or to access sensitive information until development is complete and all features are fully operational.

For now, the use of TEXTA web should be limited to testing and development purposes. It is important to remain cautious about the security of your personal data when using this browser while it is still under development.

Tasks you should **avoid**:
- Using social media;
- Using web applications (Canvas, Figma, etc.);
- Accessing sensitive accounts (banking, email, etc.).

Tasks you **can do**:
- Browsing news websites (not sensitive ones);
- Browsing entertainment websites **without logging into accounts** (YouTube, Netflix, etc.);
- Browsing information or research websites (non-sensitive).
